Class: Sprite

Sprite

new Sprite(image, spriteWidth, spriteHeight, name)

Sprite class which is at the core of the project It is a class that is used to create all of the elements on the screen apart from the background which is handled by the game object. The Sprite Class creates and stores ChildObjects which are the actual object that is created on screen. Each function here invokes a function on the child object, this allows the manipulation of multiple objects with just one function call.
Parameters:
Name Type Description
image String String reference of an image to use
spriteWidth number The width of one frame of the sprite
spriteHeight number The height of one frame of the sprite
name String The name to give the object, by default it is Sprite plus the number so far created
