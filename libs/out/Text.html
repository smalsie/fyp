Text

new Text(message, x, y, size, font, colour)

Text Class used to add text to the screen with the ability to change its values at run time.
Parameters:
Name Type Description
message String The message to display on the screen
x int X position of the text
y int Y position of the text
size String A string of the size in pixels, e.g. “34px”
font String Font of the text in string format
colour String The hex value of a colour in string format (e.g. #FFFFFF)
